# Lesson Python

## Setup

```sh
brew install python3
```

## Formatter の設定

`setting.json`

```json
  "[python]": {
    "editor.defaultFormatter": null
  },
```

[ここらへん](https://qiita.com/sin9270/items/85e2dab4c0144c79987d#autopep8)から自由に選択

```json
"python.formatting.provider": "black",
```

私は black を選択した
